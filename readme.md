## Usage
### `GET` /gif/cute
#### Endpoint
`https://cute-api.vercel.app/gif/cute`
#### Responses
|Code|Description|
|----|-----------|
|302|Successful request|
|500|Internal Server Error|
|503|Service Unavailable|
